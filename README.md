# 🚀 ShelbyDAO Dashboard

A modern Web3-style dashboard for decentralized data management, built with a clean UI and scalable architecture.

---

## ✨ Overview

ShelbyDAO Dashboard is a frontend application designed to simulate a decentralized data platform. It provides a user-friendly interface to manage users, uploads, and activity logs — inspired by Web3 and crypto dashboards.

---

## 🧩 Features

* 🔐 Simple authentication (email-based login)
* 📊 Dashboard with stats overview
* 👤 User management (add & list users)
* 📂 File upload simulation
* ⚡ Activity feed (recent actions)
* 🌙 Modern dark UI with glassmorphism
* 🧠 Structured for future Shelby integration

---

## 🛠️ Tech Stack

* **Frontend:** React
* **Styling:** Tailwind CSS
* **Backend:** Mock API (in-memory)
* **Architecture:** Modular & scalable

---
